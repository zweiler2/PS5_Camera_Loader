# PS5 Camera Loader

This makes the PS5 Camera work on a regular PC, by temporarily uploading a custom firmware to it.

## Installation

- Linux: See [Linux Instructions](README_Linux.md)
- Windows: See [Windows Instructions](README_Windows.md)
- MacOS: Could possibly work, but i do not have a Mac to test it on
